## 收获与疑问

### 今日收获

#### 1. 行列式（Determinant）
- 行列式衡量矩阵对空间做变换后，面积/体积的**缩放倍数**
- det(A) > 0 → 保持方向；det(A) < 0 → 翻转（镜像）；det(A) = 0 → 降维，不可逆
- 类比：行列式为 0 说明变换是"有损压缩"，信息丢失了

```python
A = np.array([[3, 1], [0, 2]])
np.linalg.det(A)  # 6.0，面积放大6倍
```

#### 2. 逆矩阵 / 列空间 / 零空间
- **逆矩阵 A⁻¹**：让变换"撤销"，A·A⁻¹ = I，仅 det(A) ≠ 0 时存在
- **列空间（Column Space）**：矩阵列向量线性组合所能到达的空间 = 变换的输出范围
- **零空间（Null Space）**：被压到原点的所有向量集合——"哪些输入会丢失信息"
- 零空间越大 → 矩阵越"丢信息"；零空间只有零向量 → 矩阵可逆

```python
A = np.array([[1, 2], [3, 6]])  # 列线性相关
np.linalg.det(A)  # ≈ 0，不可逆
```

#### 3. 非方阵的变换
- m×n 矩阵：n 维 → m 维映射
  - 3×2：2D 嵌入 3D（升维）
  - 2×3：3D 投影到 2D（降维）
- ML 中的意义：`nn.Linear(784, 256)` 本质就是 256×784 矩阵变换

#### 4. 点积的几何意义（Dot Product）
- 公式：a·b = |a||b|cos(θ)
- 几何含义：一个向量在另一个方向上的投影长度 × 被投影向量长度
- 点积 > 0 → 同向；= 0 → 正交；< 0 → 反向
- ML 中无处不在：余弦相似度、Attention 的 Q·K、神经网络的 wx+b

```python
a, b = np.array([1, 0]), np.array([0, 1])
np.dot(a, b)  # 0，正交
```

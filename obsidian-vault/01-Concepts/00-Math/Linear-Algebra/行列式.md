# 行列式 (Determinant)

## 一句话定义

行列式衡量一个线性变换对空间的**缩放因子**——它告诉你变换后面积（2D）或体积（3D）变成了原来的多少倍。

## 直觉/类比

想象你在 Photoshop 中对一张图片做变换：
- **det = 2**：图片面积放大到原来的 2 倍
- **det = 0.5**：图片面积缩小到原来的一半
- **det = 0**：图片被"压扁"成一条线或一个点（信息丢失，不可逆）
- **det < 0**：图片被"翻转"了（镜像），绝对值仍表示缩放倍数

软工类比：行列式就像 Git diff 的"变化量指标"——det=0 意味着信息被完全压缩（无法恢复），det≠0 意味着变换可逆（可以 revert）。

## 数学表达

**2×2 矩阵**：
$$\det\begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc$$

**几何意义**：基向量 $\hat{i}$ 和 $\hat{j}$ 变换后围成的平行四边形面积。

**关键性质**：
- $\det(AB) = \det(A) \cdot \det(B)$
- $\det(A^{-1}) = \frac{1}{\det(A)}$
- $\det(A) = 0$ ⟹ 矩阵不可逆（奇异矩阵）

## Python 代码

```python
import numpy as np

# 2x2 矩阵的行列式
A = np.array([[3, 1],
              [0, 2]])
det_A = np.linalg.det(A)
print(f"det(A) = {det_A}")  # 6.0 → 面积放大6倍

# 可视化：变换前后的单位正方形
import matplotlib.pyplot as plt

fig, axes = plt.subplots(1, 2, figsize=(10, 5))

# 变换前：单位正方形
square = np.array([[0, 0], [1, 0], [1, 1], [0, 1], [0, 0]]).T
axes[0].fill(square[0], square[1], alpha=0.3)
axes[0].set_title(f"变换前 (面积=1)")
axes[0].set_xlim(-1, 4); axes[0].set_ylim(-1, 4)
axes[0].set_aspect('equal'); axes[0].grid(True)

# 变换后：平行四边形
transformed = A @ square[:2]  # 只取前两个坐标分量
axes[1].fill(transformed[0], transformed[1], alpha=0.3, color='orange')
axes[1].set_title(f"变换后 (面积={abs(det_A):.0f})")
axes[1].set_xlim(-1, 4); axes[1].set_ylim(-1, 4)
axes[1].set_aspect('equal'); axes[1].grid(True)

plt.tight_layout()
plt.savefig("determinant_demo.png")
```

## 在 ML/DL 中的位置

```
行列式
├── 上游：矩阵、线性变换
├── 判断矩阵是否可逆 → 求解线性方程组 Ax=b
├── 协方差矩阵的行列式 → 多元高斯分布的归一化常数
├── Jacobian 行列式 → 变量替换、Normalizing Flow
└── 下游：特征值（det(A-λI)=0）、SVD
```

- **线性回归**：$X^TX$ 的行列式若为 0，说明特征之间线性相关（多重共线性）
- **PCA**：通过协方差矩阵的特征值分解降维，行列式反映数据的"展开程度"
- **生成模型**：Normalizing Flow 用 Jacobian 行列式计算概率密度的变换

## 常见面试问法

1. **行列式的几何意义是什么？** → 线性变换的面积/体积缩放因子
2. **行列式为 0 说明什么？** → 矩阵奇异，不可逆，变换将空间压缩到更低维度
3. **行列式和特征值的关系？** → $\det(A) = \prod \lambda_i$（所有特征值的乘积）
4. **为什么协方差矩阵的行列式不能为 0？** → 否则高斯分布无法归一化，说明特征存在线性相关

## 关联概念

- [[线性变换]] — 行列式描述的是线性变换的缩放效果
- [[逆矩阵]] — det≠0 是可逆的充要条件
- [[特征值与特征向量]] — 特征值的乘积 = 行列式

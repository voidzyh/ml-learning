# 线性代数 (Linear Algebra)

> 3Blue1Brown《线性代数的本质》学习笔记索引

## 📖 概念笔记

### 基础概念 (第1-2集)
- [[01-向量]] - 有大小有方向的量
- [[02-线性组合]] - 向量的"配方"
- [[03-张成空间]] - 向量能到达的所有点
- [[04-基向量]] - î(1,0) 和 ĵ(0,1)
- [[05-线性相关/无关]] - 向量组的独立性

### 核心概念 (第3-6集)
- [[线性变换]] - ✅ 已学习，矩阵即变换
- [[矩阵]] - 线性变换的表示
- [[行列式]] - 面积/体积的缩放比例
- [[逆矩阵]] - 变换的逆操作
- [[列空间]] - 矩阵的输出空间
- [[零空间]] - 变换后变成零的向量

### 进阶概念 (第7-16集)
- [[点积]] - 向量投影与对齐度
- [[叉积]] - 三维向量的"面积"
- [[特征值与特征向量]] - 变换的"主轴"
- [[特征向量]] - 保持在自身直线上的向量
- [[奇异值分解(SVD)]] - 任何变换的分解

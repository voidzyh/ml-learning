## 📝 自测题

### 1. 在机器学习中，为什么我们需要矩阵求逆？
**类型**: 场景题
**答案**: 在正规方程（Normal Equation）求解线性回归时，需要求逆来直接计算最优参数：θ = (X^T X)^(-1) X^T y。但实际常用梯度下降，因为求逆计算成本高且可能不可逆。

### 2. 用编程思维理解：什么是向量？
**类型**: 概念题
**答案**: 向量可以理解为一个一维数组，包含多个数值。在几何上，它是从原点指向空间某点的有向线段；在编程中，它是 List 或 NumPy Array。

### 3. 用 NumPy 实现矩阵乘法
**类型**: 代码题
**答案**: ```python
import numpy as np
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
C = A @ B  # 或 np.dot(A, B) 或 np.matmul(A, B)
```

### 4. 矩阵乘法的几何意义是什么？
**类型**: 概念题
**答案**: 矩阵乘法代表线性变换——对空间进行旋转、缩放、剪切等操作，把一个向量映射到另一个位置。

### 5. 点积 和叉积 的区别？
**类型**: 对比题
**答案**: 点积结果是一个标量，衡量两个向量的相似度/投影长度；叉积结果是一个向量，垂直于原来两个向量构成的平面，表示面积和方向。
